# Contributing to Claude Skills Collection

Thank you for your interest in contributing! This is a curated collection of skills from various repositories.

## How to Contribute

### Adding New Skills

This collection consolidates skills from multiple source repositories. If you have a new skill to contribute:

1. **Check if it belongs in an existing source repository** — Most skills should be contributed to their original source repository first:
   - [anthropics/claude-code](https://github.com/anthropics/claude-code)
   - [anthropics/anthropic-quickstarts](https://github.com/anthropics/anthropic-quickstarts)
   - [trailofbits/claude-code-skills](https://github.com/trailofbits/claude-code-skills)
   - [Yeachan-Heo/oh-my-claudecode](https://github.com/Yeachan-Heo/oh-my-claudecode)

2. **For new source repositories** — If you have a new repository of skills that should be included:
   - Open an issue proposing the repository
   - Include a description of the skills it provides
   - Explain why it should be included

3. **For improvements to existing skills** — Please contribute directly to the source repository

### Reporting Issues

If you find an issue with a specific skill:
1. Check which source repository it came from (see README.md)
2. Report the issue in that repository
3. Optionally, open an issue here to track it

### Documentation Improvements

Documentation improvements are welcome:
- Fix typos
- Improve clarity
- Add missing examples
- Update outdated information

## Development Workflow

1. Fork this repository
2. Create a branch for your changes
3. Make your changes
4. Submit a pull request

## Code of Conduct

Be respectful and constructive in all interactions.
