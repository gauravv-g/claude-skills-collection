## How to Use

### Using /auto Command

Simply type `/auto` followed by your task:
```
/auto add dark mode to the app
/auto fix the authentication bug
/auto create a PR for the new feature
```

### Using oh-my-claudecode

**Setup:**
```bash
/setup
/omc-setup
```

**Autopilot:**
```
/autopilot "build a REST API for managing tasks"
```

**Deep Interview:**
```
/deep-interview "I want to build a task management app"
```

**Team Mode:**
```
/team 3:executor "fix all TypeScript errors"
```

**Ask External Providers:**
```
/ask codex "review this patch"
```
